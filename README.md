# Smart Urban Environmental Risk Prediction System

Automated ML system that:
- stores live city data in MySQL
- retrains models automatically using Python
- triggers retraining via Node.js API
- tracks model accuracy over time

## Tech Stack
- Python (ML models)
- Node.js (API)
- MySQL (database)
- Scikit-learn
- SQLAlchemy

## Features
- Auto retraining
- Real-time predictions
- Performance monitoring
- Clean modular architecture

## How to Run
1. Start MySQL
2. Run backend server
3. Send POST data
4. Get predictions
